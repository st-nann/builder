# Template Builder

### How to use plugin
```
1. Add Dependencies on package.json
    "builder": "git+ssh://git@bitbucket.org:3dsinteractive/pam-builder.git"
2. Run Command
    $ npm install builder --save
    $ yarn add builder
3. Import Component
    import 'builder'
```

## Project setup
```
yarn install
```
